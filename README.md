# oliveJS

## How to install?
---
Just copy and paste this:

```git clone https://github.com/olvrb/oliveJS && cd oliveJS && sudo npm install && nodemon -d 1 app.js```

If you just want this bot on your server, you can add it using [this link.](https://discordapp.com/oauth2/authorize?client_id=374188904309456896&scope=bot&permissions=8)
# Shitty Documentation

### Functions
---
- sendEmbedded 
    - What is it?
        - Easily send embedded messages.
    - Arguments
        - int: embed color in decimal.
        - string message to send.
- toBin
    - What is it?
        - Easily convert any string to binary.
    - Arguments
        - string: input string to convert to binary.
- fromBin
    - What is it?
        - Easily convert any binary string to a regular string.
    - Arguments
        - string: input string to convert to a regular string.
- to64
    - What is it?
        - Easily convert any string to a base 64 string.
    - Arguments
        - string: input string to convert to a base 64 string.
- from64 
    - What is it?
        - Easily convert any base 64 string to a regular string.
    - Arguments
        - string: input string to convert from a bse 64 string.
- getImage.getImagesArray
    - What is it?
        - Easily get an array of images from google images.
    - Arguments 
        - string: keyword to search for.
- Animate
    - What is it? 
        - Easily animate multiple strings.
    - Arguments
        - string: string to animate word by word.
        - int: how long it waits between each edit.
